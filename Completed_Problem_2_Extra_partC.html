In this notebook we are going to change our density model to account for dark matter. We are doing this using the Burkert profile. The equation for the Burkert profile is as follows: $\rho_{dm}(r) = \frac{\rho_{1}}{(1 + \frac{r}{r_1}) (1 + (\frac{r}{r_1})^2) }$

Where $\rho_{1}$ is the central density of the dark matter(how much dark matter is the center) and $r_1$ is the scale radius of the dark matter halo(how spread out the dark matter is from the core). The $r$ in this equation is just the radius from the center. Later we will intergrate this equation to get the mass of the dark matter within a given radius.

In this next chunk we define our model to calculate velocity at a given radius. The velocity profile function stays the same as before being defined as: $v(r) = \sqrt{\frac{GM_{\text{in}}(r)}{r}}$ The M_in is a function to find the mass within a given radius and it does not change either being defined as: $M_{\text{in}}(r) = \int_0^r 4\pi \rho(r') r'^2 \, dr'$ The density profile(p) does need to change to account for the new mass in the burkert profile. The new density profile will take new arguments of p1 and r1 is defined as:

$\rho(r) = \begin{cases} \rho_0+\frac{\rho_{1}}{(1 + \frac{r}{r_1}) (1 + (\frac{r}{r_1})^2)} , & \text{if } r < r_0 \\ \frac{\rho_{1}}{(1 + \frac{r}{r_1}) (1 + (\frac{r}{r_1})^2)}, & \text{otherwise} \end{cases}$

In[]:
def p(r,r0,p0,p1,dark_r1):
    #Dark mass
    part1 = 1+(r/dark_r1)
    part2 = 1+(r/dark_r1)**2
    dark_mass = p1 / (part1*part2)

    if r<r0:
        return p0+dark_mass
    else:
        return dark_mass

#new parameters have been introduced to this model to account for the burkert profile
def M_in(r,p0,r0,p1,dark_r1):
    lst=[]
    for r1 in r:
        #Light mass
        intergrand = lambda r_prime: 4*np.pi*p(r_prime,r0,p0,p1,dark_r1)*(r_prime**2)
        light_mass, _ = quad(intergrand,1e-6,r1)

        lst.append(light_mass)
    
    return np.array(lst)

#Does not change with the new parameters except passing them into the M_in function
def velocity_profile(r,p0,r0,p1,dark_r1):
    G = 6.6743e-11
    M = M_in(r,p0,r0,p1,dark_r1)
    return np.sqrt(G*M/r)

Our walker function still mostly resembles the walker functions in previous notebooks. The only difference is that we are now passing in the new arguments of p1 and r1. Our prior makes sure that these values will never be negative.

In[]:
def ln_prior(params):
    p0,r0,p1,dark_r1 = params
    if p0>0 and r0>0 and p1>0 and dark_r1>0:
        return 0
    return -np.inf

#Finds how well the model fits the data
def ln_likelihood(params,r,V_obs,V_err):
    p0,r0,p1,dark_r1 = params
    v_model_pred = velocity_profile(r,p0,r0,p1,dark_r1)
    return -0.5*np.sum((V_obs-v_model_pred)**2/V_err**2)

#Finds how likely the model is
def ln_pdf(params,r, V_obs, V_err):
    p=ln_prior(params)
    if p==-np.inf:
        return -np.inf
    return p + ln_likelihood(params,r,V_obs,V_err)
